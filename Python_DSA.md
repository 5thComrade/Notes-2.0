### Linked Lists

A list(array in JavaScript) has its items stored next to each other in the memory, hence we can use index's to retrive items from the list.

When it comes to Linked Lists, the items are all scattered all over the memory. There is a `head` variable pointing to the first item and a `tail` variable pointing to the last item. 
Each item in the linked list is then pointing to each other and the final item points to none.
